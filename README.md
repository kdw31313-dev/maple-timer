# 개인용 메이플 사냥 타이머 (Maple Hunting Helper Timer)

웹 브라우저의 화면 공유(Screen Capture API) 및 Canvas 영상 처리 알고리즘을 기반으로 동작하는 **개인용 메이플 사냥 보조 타이머**입니다.

---

## 🌟 주요 특징 및 원칙

1. **100% 클라이언트 브라우저 전용 (보안 & 개인정보)**
   - 게임 메모리 탐색, 패킷 감지, 클라이언트 파일 접근이 전혀 없습니다.
   - 키보드/마우스 자동 입력(매크로) 기능이 전혀 없으며, 실제 입력 조작은 100% 사용자가 직접 진행합니다.
   - 화면 데이터는 서버로 전송되지 않으며, 사용중인 PC의 브라우저 캔버스 내에서만 실시간 분석됩니다.

2. **화면 인식 알림 시스템**
   - **룬 등장 감지**: 사용자가 룬이 생성되는 화면 영역(ROI)을 드래그하여 지정하면, 2~3회 연속 색상/차분 변화를 확인하여 오탐 없이 알림을 발생시킵니다. (룬 지속 중 중복 알림 방지 쿨다운 적용)
   - **4종 팝업 / 거짓말 탐지기 감지**: 일반 문구 팝업, 정답 문장 선택형, 비올레타 미니게임 창, 투명 도형형 팝업의 갑작스러운 출현을 감지하여 알림을 발생시킵니다.

3. **수동 사냥 타이머**
   - **경험치 쿠폰 타이머**: 15분 / 30분 / 60분 프리셋 지원. 60초 전, 30초 전, 종료 직후 사운드 & 음성 알림.
   - **솔 야누스 주기 타이머**: 70초 / 80초 주기 선택. 재사용 5초 전 프리 알림. `Spacebar` 단축키 원클릭 재시작 지원.

4. **Picture-in-Picture (PiP) 플로팅 HUD**
   - 모니터 1개 사용자도 메이플 게임 화면 위에 작은 타이머 및 룬/팝업 상태창을 상시 위에 띄워 두고 사냥을 즐길 수 있습니다.

5. **다양한 알림 모드**
   - Web Audio API 합성음 (차임벨, 비프음, 룬음, 사이렌)
   - 한국어 음성 안내 (TTS: "룬이 출현했습니다!", "경험치 쿠폰 종료 1분 전입니다")
   - 화면 깜빡임 (Visual Flash Overlay)

---

## 🚀 사용법 가이드

1. `index.html` 파일을 웹 브라우저(Chrome, Edge 권장)로 엽니다.
2. 상단의 **`▶ 게임 창 공유 시작`** 버튼을 누른 후 메이플스토리 창을 선택합니다.
3. **영역 지정**:
   - `룬 감지 영역 드래그 지정` 버튼을 누르고 화면 미리보기 상에서 룬이 등장하는 지형 범위를 드래그합니다.
   - `팝업 감지 영역 드래그 지정` 버튼을 누르고 팝업이 뜨는 중앙 범위를 드래그합니다.
4. 사냥 시작 시 경험치 쿠폰 및 솔 야누스 타이머 버튼을 누릅니다. (`Spacebar`로 야누스 타이머 즉시 시작 가능)
5. 필요 시 **`📺 PIP 플로팅 창`** 버튼을 눌러 게임 화면 위에 오버레이를 띄웁니다.
